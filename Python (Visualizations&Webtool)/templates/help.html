Getting started

Welcome to Proteostasis Profiler web tool. In this section we briefly explain how you can use this tool.
For security reasons, only authenticated users can use this tool. so in case you haven't already, you can use the links on the top left corner of the screen to "sign up" and "log in". Signing up is free for any person with a valid email address.
After signing in, you are directed to the main dashboard screen. In the dashboard you go through four steps to choose how you want to use the tool.

  1. In step one you choose in what level of details you want to analysis the data. "PN" or Proteostasis Network analyses the data in a general level, with only having six subgroups considered when studying the whole PN, while "Chaperome" goes into more detailed subgroups existing in the Folding.
  1. In step two you choose in what disease group you want to analyze proteostasis. For now the available options are "Cancer(TCGA)" and "Neurodegenerative" diseases.
  1. In step three you can choose the tissues (or diseases) to analyse. For choosing the tissues you first have to click on "Disease Selector Wizard", a new window will open with all the available tissue in the disease group you chose in step 2. You can choose one or more tissues. Clicking the "go to the next step" button will send you to step 4. Remember that if you want to compare proteostasis in few tissues or diseases, select all of them in this step.
  1. In step 4, you choose how you want the results to be shown. there are four options:
    1. Heatmap plot: shows the results of "GSA" or "Gene Set Analysis" on the chosen tissues in the last step (More info about the method in this study). If less than 3 tissues were chosen,all the available tissues are shown.
    1. Polar map: shows M-score results on the chosen tissues (More info about the method in this study). If more than one tissue is chosen, each tissue are plotted on one Polar plot individually.
    1. Meta-interactome: shows M-score results on the chosen tissues, mapped on Meta-interactome of the Protein Protein interactions network. (More info about the method in this study). If more than one tissue is chosen, each tissue are plotted on one Meta-interactome plot individually.
    1. 3D topography: shows M-score results on the chosen tissues, mapped on Meta-interactome of the Protein Protein interactions network in 3D space. (More info about the method in this study). If more than one tissue is chosen, each tissue are plotted on one 3D topographic plot individually.
After passing all four steps, the results are shown. You can use the "control bar" on top of the screen to change the "Plot Type" or change the tissues shown using the "Disease Selector Wizard".
